Screenshots

Home loading

Screenshot of how the application looks when it is opened. It is the first view that all users will have of the application.

Settings

Basic configuration screen of the application. In this you can configure the following characteristics:

  • Inventory:
    Disables inventory at will
  • Run inventory now:
    Scans the device and creates the inventory
  • Global settings:
    Takes you to the global settings screen

Global Settings

In this screen you can configure the following parameters:

  • Notifications:
    Allows you to define the address of access server
  • Server address:
    Allows you to change the access address to the server
  • Tag:
    Allows you to define a tag
  • HTTP authentication credentials:
    In this section you need to enter your user and password of Flyve MDM
